# 驾驶视频资源清单

这份文档收集适合重新练车时看的公开视频资源，重点覆盖：转弯、掉头、起步、停车、侧方停车、倒车入库、科目二、科目三。

使用建议：视频只用来建立画面感，真正上路仍然要找安全场地和教练陪练。不同城市、不同考场、不同车型的点位会有差异，考试细节以当地驾校和考场要求为准。

## 1. 推荐观看顺序

如果很久没开车，建议按这个顺序看：

1. 起步、停车、方向盘、低速控制。
2. 转弯、右转、左转、掉头。
3. 变道、靠边、路口观察。
4. 倒车入库、侧方停车。
5. 科目二全项目。
6. 科目三全流程。

不要一上来只看倒库点位。先把“眼睛看哪里、脚怎么备刹、方向什么时候回正”看明白，再看具体项目会更稳。

## 2. 新手实战：转弯、掉头、起步停车

### 转弯掉头怎么打方向

[转弯掉头怎么打方向？沉浸式体验教学！](https://www.bilibili.com/video/BV1sb4y1e7c8/)

适合看：

- 转弯时方向盘打多少。
- 掉头时什么时候多打方向。
- 转弯后什么时候回正。
- 第一视角理解车头走向。

观看重点：

- 不要只看方向盘，要看车头和出口。
- 低速可以多打，高速只能小幅修。
- 车头快正时提前回正。

### 小区转弯问题

[小区转弯问题，这个视频详细讲解](https://www.youtube.com/watch?v=VIU-fT0H_AQ)

适合看：

- 小区窄路怎么转弯。
- 低速时怎么观察车身和路边。
- 新手在窄路转弯时怎么避免剐蹭。

观看重点：

- 转弯前先慢下来。
- 眼睛看出口，不要只盯车头。
- 右转时特别注意右侧盲区和路边行人。

### 左转弯实战带教

[左转弯保姆级带教](https://www.youtube.com/watch?v=9j3hhYZYOIY)

适合看：

- 左转怎么观察对向直行车。
- 什么时候进入路口。
- 什么时候开始打方向。

观看重点：

- 左转先让对向直行。
- 不抢黄灯。
- 转弯后进入正确车道并及时回正。

## 3. 科目二：倒车入库、侧方停车、曲线、直角

### 科目二全项目教程

[科目二驾考教程详解：考试项目、评判标准、基础操作](https://www.bilibili.com/video/BV1H4411G7gB/)

这个视频是合集，包含：

- 上车准备。
- 倒车入库。
- 侧方停车。
- 曲线行驶。
- 直角转弯。
- 坡路定点停车与起步。
- 窄路掉头等部分地区项目。

适合看：

- 建立科目二整体框架。
- 了解每个项目大概怎么走。
- 复习考试项目名称和顺序。

观看重点：

- 倒库和侧方不要只背点位，要理解车尾方向。
- 曲线和直角重点是低速、看边线、及时回正。
- 坡道定点和起步如果你以后主要开电车，可以只了解考试逻辑，日常驾驶更重视刹车、AUTO HOLD 和坡道防溜。

### 科目二考试视频全程

[科目二考试视频全程，教练讲解各项操作及考试要领](https://www.bilibili.com/video/BV1gs411a7Cg/)

适合看：

- 连续看完整考试流程。
- 理解科目二各项目之间如何衔接。
- 看考试时车速应该多慢。

观看重点：

- 低速比快更重要。
- 每个项目开始前先摆正车身。
- 打方向后要记得回正。

### 科目二倒车入库调整

[科目二倒车入库，一边宽一边窄怎么办？右倒库](https://www.bilibili.com/video/BV1564y1r7nZ/)

适合看：

- 倒库入库后一边宽一边窄怎么修。
- 右倒库方向修正。
- 通过后视镜判断车身位置。

观看重点：

- 倒车时“车尾想去哪，方向盘就往哪打”。
- 车身快正时提前回方向。
- 不要只背一个点位，学会看后视镜修正。

### 科目二侧方停车

[科目二侧方停车视频教学，一镜到底](https://www.bilibili.com/video/BV1FYBYBWEpy)

适合看：

- 侧方停车完整入库过程。
- 低速倒车时怎么看后视镜。
- 入库后怎么出库。

观看重点：

- 车速一定要慢。
- 入库看车尾和库线关系。
- 出库前打灯、看前后左右。

### 侧方停车多种出库方法

[科目二侧方停车六种出库方法](https://www.bilibili.com/video/BV1iX4y1f7LM/)

适合看：

- 侧方停车出库。
- 不同车身位置下怎么修方向。
- 避免只会一种死点位。

观看重点：

- 出库先看前方和左后方。
- 车头够空间再打方向。
- 不确定就停一下再判断。

## 4. 科目三：起步、变道、直线、路口、靠边停车

### 科目三全流程合集

[科目三驾考教程详解：考试项目和评判标准](https://www.bilibili.com/video/BV1n4411G7Li/)

这个合集包含：

- 上车准备。
- 模拟夜间灯光。
- 起步。
- 变更车道。
- 直线行驶。
- 通过公交站、学校、路口、人行横道。
- 会车、超车、掉头。
- 加减挡。
- 路口左转、右转。
- 靠边停车。
- 夜间行驶。

适合看：

- 系统了解科目三考什么。
- 复习每个项目的动作顺序。
- 看起步、变道、靠边停车这些和日常驾驶很接近的动作。

观看重点：

- 起步前看镜、打灯、回头观察。
- 变道前看镜、打灯、再看盲区。
- 通过路口、人行横道、学校、公交站要提前减速和观察。
- 靠边停车先看右镜和右侧盲区。

### 科目三 20 小节教程

[最全最详细的科目三教程（20小节）](https://www.bilibili.com/video/BV1C54y1m7b3/)

适合看：

- 科目三细节拆分。
- 起步动作。
- 加减挡。
- 变更车道。
- 直线行驶。
- 靠边停车。

观看重点：

- 科目三不是只考操作，也考观察和安全意识。
- 每次改变方向前都要有灯光、观察、确认。
- 直线行驶不是死抓方向盘，而是看远处、小幅修正。

### 科目三标准教程

[安徽省驾考 C1 科目三标准教程](https://www.bilibili.com/video/BV1qU4y1x7sM/)

适合看：

- 标准化路考流程。
- 起步、直线行驶、加减挡、变更车道、靠边停车。
- 了解地方考场路线视频通常怎么讲。

观看重点：

- 地方考场路线只能参考，不要照搬到其他城市。
- 真正通用的是观察、打灯、减速、让行、车距。

### 科目三教学合集

[科目三教学](https://www.bilibili.com/video/BV1Zq4y1w7X4/)

适合看：

- 模拟夜间灯光。
- 上车准备。
- 起步。
- 直线行驶。
- 变更车道。
- 通过路口。
- 路口左转弯、右转弯。

观看重点：

- 按项目拆开看，每天看一两个，不要一次全塞进脑子。
- 起步、变道、靠边停车最值得反复看。

## 5. 倒车和停车专项

### 现实停车场倒车入库

[专业教练教你：零基础学会停车场倒车入库](https://www.youtube.com/watch?v=8dB9jgCrYaw)

适合看：

- 现实停车场倒车入库。
- 不只看驾校点位，而是看车位和车身关系。
- 商场地库、普通车位怎么倒。

观看重点：

- 倒车影像只能辅助，仍要看左右后视镜。
- 车尾想去哪，方向就往哪打。
- 车身快正时提前回方向。

### 倒车方向练习

[方向盘容易打懵，四个倒车练习](https://www.youtube.com/watch?v=NnyxSK58Vdc)

适合看：

- 倒车左右分不清。
- 低速倒车时如何练方向感。
- 怎么避免忘记方向盘打了多少。

观看重点：

- 先在空旷场地练，不要一上来进复杂地库。
- 倒车时速度越慢越好。
- 不确定就停住，看镜子和车身，再修。

### 倒车入库左右分不清

[倒车入库打方向盘，左右分不清的看看](https://www.bilibili.com/video/BV1Hv4y1A7Nz/)

适合看：

- 倒车时方向盘左右关系。
- 车尾和方向盘的对应关系。
- 倒车入库基础理解。

观看重点：

- 想让车尾往哪边去，方向盘就往哪边打。
- 车尾往左，方向往左。
- 车尾往右，方向往右。

## 7. 复习清单

看视频时不要只看热闹，每个视频都问自己这几个问题：

- 这个动作前，眼睛先看哪里？
- 右脚是在电门上，还是已经备刹？
- 有没有先打转向灯？
- 有没有看后视镜和盲区？
- 方向盘什么时候开始打？
- 车头快正时有没有提前回方向？
- 遇到行人、电动车、对向车时，谁让谁？
- 如果我在现实路上不确定，应该在哪里停下来观察？
