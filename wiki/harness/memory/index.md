# Memory

Memory 层记录当前任务状态、跨 session 知识、记忆召回和记忆持久化。

它关注哪些信息应该短期保留、哪些知识值得跨 session 保存、何时检索、如何更新和删除。Memory 不等于把所有历史塞回 prompt；关键是 scope、provenance、retrieval 和 persistence policy。
