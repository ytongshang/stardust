# Multi-Agent

Multi-agent 层记录子 agent、后台 agent、agent 间通信、team 结构和协调策略。

它关注多个 execution subjects 如何拆分任务、通信、并行运行、合并结果和共享 workspace。这里不把“多开几个 chat”当作概念，重点看 task contract、lifecycle、communication protocol、coordination 和 merge。
