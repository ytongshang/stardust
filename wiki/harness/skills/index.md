# Skills

Skills 层记录可按需发现、加载、落地和版本化的能力包。Skill 可能指导工具使用，也可能只是文档、脚本、模板或资源集合；它不是普通 memory，也不等同于 tool。

它关注能力包的生命周期：如何被发现、何时加载完整说明、资源如何 materialize、版本如何 pin 或更新。外部项目如果只是把 prompt 模板叫做 skill，不一定值得写；重点看它是否形成可组合、可缓存、可版本化的 capability package。
