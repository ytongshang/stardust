# Guardrails

Guardrails 层记录 Permissions、Approvals & HITL、Sandbox Policy 和 application/security risk control。

它关注 agent 能不能做、什么时候需要确认、在哪个隔离环境做，以及遇到风险时如何降级或阻断。Guardrails 不是 prompt 里的安全提醒，而是 runtime 和 product layer 必须强制执行的 policy。
