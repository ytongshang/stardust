# Observability

Observability 层记录 trace、replay、eval 和失败分类。

它关注 harness 是否可解释、可复现、可评估。一个 agent 失败后，Observability 应该能回答发生了什么、能否 replay、属于哪类 failure，以及改动后是否通过 eval 变好了。
