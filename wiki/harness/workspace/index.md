# Workspace

Workspace 层记录 agent 可读写、可执行、可发布、可合并的工作区与产物体系。

它关注 agent 操作的对象世界：files、artifacts、snapshots、sandbox、branch/merge 和 preview。这里不记录某个具体产物的内容，而是记录 workspace backend 和 artifact lifecycle 如何支撑可靠执行。
